Compiler & Run Java in Sublime

Posted on 2019-04-27 | In BackEnd

有的時候不想開IDE的情況下測一些很小的東西，這個時候Sublime這種輕巧的編輯器就派上用場了，但這時候想要使用它的腳本工具一口氣編譯執行，但他內建只有編譯，沒有執行，所以就自己寫一個小腳本來執行。

Tools > Build System > New Build System...

在編譯執行前先選擇編譯工具，這個時候選剛才新建的小工具，Build System > runJava，接著Ctrl+B執行，Console中就有我們要的結果了。
