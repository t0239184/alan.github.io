nohup 指令 - 背景執行指令

於背景執行Linux命令直到完成後才會結束, 不會影響使用者的正常工作。

nohup = “no hangup”

nohup 預設會將指令的輸出放到 nohup.out 的檔案,如果需要自訂重新導向的檔案, 可以用下面方法重新導向到檔案

這樣就會將指令的輸出放到 nohup.log 內。

Run process on backgrund

$ nohup java -jar project.jar –server.port=port &

See process log
